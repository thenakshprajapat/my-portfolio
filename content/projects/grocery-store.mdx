This project simulates a grocery store experience right from the command line.  
Users can:
- Add or remove items from their cart  
- View total cost and item summary  
- Check available inventory  
- Exit with a final bill  

**Concepts Used:** `sets`, `loops`, `conditionals`, `functions`, `user input`.

**Highlight:** Built entirely using Python core concepts, without external libraries.
