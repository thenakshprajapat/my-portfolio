This project converts inconsistent or unclean strings into standardized, readable text formats.  
Users can:
- Capitalize sentences properly  
- Trim unnecessary spaces  
- Format output dynamically using f-strings  

**Concepts Used:** `string methods`, `loops`, `type conversion`, `f-strings`.
