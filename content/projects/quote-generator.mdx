A simple yet fun project that prints random quotes each time it runs.  
Users can:
- Fetch a random quote from a list  
- Add their own quotes  
- Generate multiple random quotes  

**Concepts Used:** `lists`, `random module`, `functions`.

**Highlight:** A great project for beginners to understand list manipulation and randomness.
